# NextSchoolAI — Guide d'Utilisation

> **Version** : 1.1 — Le guide complet pour utiliser la plateforme NextSchoolAI.

---

## 1. Présentation

**NextSchoolAI** est une plateforme éducative camerounaise qui permet de :
- 📚 **Consulter** des documents de cours, épreuves et livres
- 🤖 **Interagir** avec une IA pédagogique pour poser des questions sur les cours
- 📝 **Générer** des QCM automatiquement à partir des documents
- ⬇️ **Télécharger** les fichiers PDF

---

## 2. Connexion et Navigation

### Se connecter
1. Allez sur la page d'accueil
2. Cliquez sur **Connexion** et entrez vos identifiants
3. Vous arrivez sur la **Bibliothèque**

### Barre latérale (Sidebar)

| Menu | Description |
|------|-------------|
| 📚 Bibliothèque | Parcourir tous les documents |
| 📋 Mes QCM | Historique de vos QCM passés |
| 📊 Tableau de bord | Statistiques d'activité |
| ☁️ Uploader | Ajouter un document (enseignants) |
| 📁 Mes documents | Gérer vos uploads (enseignants) |

### Recherche avancée
- Cliquez sur la barre de recherche ou appuyez sur **⌘K** (Ctrl+K)
- Filtrez par **niveau**, **matière**, **type de document** et **année**

---

## 3. Page de Document

Quand vous ouvrez un document, vous voyez :

### En-tête
- **Titre** du document, **matière**, **classe**, **année**
- Bouton **Télécharger le PDF**
- Bouton **Générer QCM** (pour créer un quiz automatiquement)

### Zone de Chat IA ✨

C'est la fonctionnalité principale ! Un assistant IA est directement intégré dans la page du document.

#### Comment l'utiliser

1. **Le document est automatiquement attaché** — l'IA a accès au contenu complet
2. **Écrivez votre question** dans la zone de texte en bas
3. **Appuyez sur Entrée** ou cliquez sur le bouton d'envoi
4. **L'IA répond** avec du texte formaté (titres, listes, code, formules maths)

#### Suggestions rapides

Au démarrage, 4 boutons de suggestion sont proposés :

| Suggestion | Ce que fait l'IA |
|-----------|-----------------|
| 📝 Résumé complet | Génère un résumé structuré du document |
| 💡 Concepts clés | Explique les notions principales |
| 🔢 Formules importantes | Liste les formules mathématiques/physiques |
| 📖 Fiches de révision | Crée des fiches condensées pour réviser |

#### Exemples de questions que vous pouvez poser

- *"Fais-moi un résumé de ce cours"*
- *"Explique-moi le chapitre 3"*
- *"Donne-moi un exemple de code Python pour la POO"*
- *"Quelles sont les formules importantes ?"*
- *"Crée des exercices d'entraînement"*
- *"Explique la différence entre X et Y"*

#### Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| **Entrée** | Envoyer le message |
| **Shift + Entrée** | Nouvelle ligne |
| **⌘K / Ctrl+K** | Ouvrir la recherche |

#### Bouton 🔄 Nouvelle conversation
Cliquez sur l'icône en haut à droite du chat pour effacer l'historique et recommencer.

---

## 4. Générer un QCM

1. Sur la page du document, cliquez sur **Générer QCM**
2. L'IA analyse le document et crée un QCM de 10 questions
3. Répondez aux questions
4. Consultez votre score et les corrections détaillées

---

## 5. Évaluer un Document

- En bas de la page, donnez une **note de 1 à 5**
- Ajoutez un **commentaire** (optionnel)
- Cliquez sur **Évaluer**

---

## 6. Pour les Enseignants

### Uploader un document
1. Allez dans **Uploader** depuis la sidebar
2. Remplissez le formulaire : titre, matière, classe, fichier PDF
3. Le document est analysé par l'IA automatiquement
4. Après validation humaine, il sera publié dans la bibliothèque

### Gérer vos documents
- **Mes documents** : voir tous vos uploads avec leur statut

---

## 7. FAQ

**Q : L'IA met du temps à répondre ?**
> C'est normal, l'IA analyse le document complet. Attendez quelques secondes.

**Q : L'IA donne une réponse incorrecte ?**
> L'IA peut faire des erreurs. Vérifiez toujours les informations importantes avec votre cours.

**Q : Comment changer de document dans le chat ?**
> Naviguez vers un autre document. Chaque document a son propre chat.

**Q : Puis-je poser des questions hors du sujet du document ?**
> L'IA est optimisée pour répondre sur le contenu du document attaché. Les questions hors-sujet peuvent recevoir des réponses moins précises.
